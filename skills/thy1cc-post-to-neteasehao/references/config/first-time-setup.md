# First-time Setup for `thy1cc-post-to-neteasehao`

Create one of these files:

- `.thy1cc-skills/thy1cc-post-to-neteasehao/EXTEND.md`
- `$HOME/.thy1cc-skills/thy1cc-post-to-neteasehao/EXTEND.md`

Example:

```md
chrome_profile_path: /Users/you/.local/share/neteasehao-browser-profile
content_manage_url: https://mp.163.com/article/manage
default_max_pages: 2
slow_mode_ms: 1200
```

Notes:

- `chrome_profile_path`: use a stable profile that keeps login state.
- `content_manage_url`: set this if your account lands on a different management route.
- `slow_mode_ms`: keep >= 1000 for lower risk.
